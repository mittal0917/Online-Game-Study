```

해당 자료는 교육 및 학습을 위해서만 자유롭게 사용해주시면 감사하겠습니다.
자료에 사용된 모델이나 폰트, 이미지 등을 상업적 용도로 무단 사용하는 것을 금지합니다.
```

## 1. 게임 \<Ultimate Cart Fights\> 소개

친구와 함께 즐길 수 있는 온라인 범퍼카 게임입니다.  
상대의 카트에 힘껏 부딪쳐 탈락시켜보세요!  
해당 게임은 Photon Fusion 2를 활용하여 제작되었습니다.  

### 1-1. 게임 규칙

```
1. WASD 혹은 방향키로 자신의 카트를 조종합니다.
2. Q를 눌러 대시를 사용할 수 있습니다.
3. 상대방의 카트에 부딪쳐 데미지를 줄 수 있습니다.
4. 빠른 속도로 부딪칠수록 더 큰 데미지를 줄 수 있습니다.
5. 카트의 누적 데미지가 999.9%에 도달하면 해당 카트는 탈락합니다.
6. 최후에 남은 1인이 승자가 됩니다.
```

## 2. 스터디 \<온라인 게임 스터디\> 소개

<br>

> [스터디 문서 링크](https://star-shark-54e.notion.site/1b01ea5c2288801696b1ec2228d864c5?v=4bc123c6cc7e4452ac3977747c59e36a, "스터디 문서 링크")

<br>

\<온라인 게임 스터디\>는 게임 제작 동아리 INGAME에서 진행되는 프로그래밍 스터디 입니다.  
해당 스터디는 Photon Fusion 제품을 사용하여 서버 구축 없이 3D 온라인 게임을 제작하는 방법을 배우는 스터디입니다.  
네트워크나 OS와 같은 CS 지식들 없이도 본 스터디를 따라하실 수 있도록 구성하였습니다.  

본 스터디는 Unity와 C#에는 익숙하지만 온라인 게임 제작에는 익숙하지 않은 분들을 대상으로 제작되었습니다.  
따라서 Unity와 C#의 기본적인 내용은 분량 상 설명이 생략되어 있습니다.  
이 점 참고바랍니다.  

### 2-1. 스터디 일정

스터디는 다음과 같이 6주 정도의 분량으로 진행됩니다.  

| 주차 | 학습 내용 |
|:---:|:---|
| 1주차 | 프로젝트 설정하기 |
| 2주차 | Photon Fusion 시작하기 |
| 3주차 | 방 목록이 보이는 로비 만들기 |
| 4주차 | 커스터마이징이 가능한 게임룸 만들기 |
| 5주차 | 로딩 기능 구현하기 |
| 6주차 | 온라인으로 플레이어 움직이기 |

### 2-2. 스터디 진행 방식

\<온라인 게임 스터디\>는 매주 월요일마다 INGAME 디스코드 내 "온라인 게임 스터디" 채널에 공지와 함께 스터디 문서를 제공할 예정입니다.  
해당 문서를 통해 학습을 진행한 후 기능 작동 시연 영상을 녹화하여 INGAME 카페의 스터디 과제 제출 게시판에 올려주시면 됩니다.  
만일 스터디 진행 중 궁금한 사항이나 오류가 있다면 디스코드 내 "온라인 게임 스터디 Q&A"에 해당 내용을 작성하여 올려주시기 바랍니다.
